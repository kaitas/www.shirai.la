初っ端からPlayCanvasが実行できない時の話

こんにちは東田です．

本日は新四年のゼミ日でした．

全員の進捗報告を行った後に，プロジェクターに接続した途端にPCがブルースクリーンを吐き出した望月くんを中心とし，Playcanvasを用いた多重化のワークショップを行いました．

基礎的なグラフィクス数学の話から始まり，PlayCanvasを用いた簡単な画像変化デモを作る事になったのですが…

 

新規作成！NEWを押していざEDIT．

この時点でなんかおかしい．

本来の実行画面ですと真ん中上段にシーンが表示されているはずなのですが…無い．

おかしいなあ……不安を感じつつ実行してみる．

 

なんでや．

 

Uncaught TypeError:Cannot read property ‘applySceneSettings’ of null

……未捕獲のタイプエラー？（違います）

 

 

 

applySceneSettingってなんじゃらほい．作成も削除もした覚えはないぞ…？

プロジェクトの作成に失敗したのか？ともう一度作り直しても，

まさかメモリが足りないのかと他のアプリケーションを終了してから作り直しても，

何故か表示されない．おまけにググっても出ない．ちょっとこれはお手上げです…．

 

ということで，某先輩にお尋ねしてみました．ヘルプミー！

 

即答だったよ！わーい！ありがとうございます！

 

ということでこちらを参考に有効化します．

うろ覚えでは有るのですが，初期設定では有効だったものを自分で切ってしまったのだと思います．（当時，しょっちゅうChromeのメモリ不足が表示されるので，その解決策として設定を変えたのでした）

 

実行してみると…

おっ

表示された！！！！！！！

わーい嬉しい．

前述のようにググっても出なかったので，

もし自分のように引っかかる人がいてお手上げになった際の道標にしておこうと思います．

……え？そんな間抜けはそうそう居ない？
