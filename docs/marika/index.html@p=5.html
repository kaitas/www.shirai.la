お引っ越し

こんにちは東田です．

三年セミナーブログからお引っ越しするような形で新しいブログになりました．

個人ブログです！

 

 

これまでの経験上不安しか無い！！（三日坊主どころか一日坊主になる人）

せめて一年頑張れれば…と思いますが，初心が長続きすることを祈るばかりです．

 

さて，このブログで何を扱うのか全く決めていなかったりします．

完全に個人でつくったブログの方もあるので使い分け出来たらとは思うのですが，うーむ．

 

展示や発表に参加した時のレポはこちらでやっていきたいと思っています．

その他研究室の出来事や開発で詰まった点を気分転換がてらに書く癖が付けられたら…なあ…（遠い目）

とりあえずはこの記事をテスト投稿と言う形で利用し，デザインを選びたいですね！

なかなか自分の要求とマッチするテンプレートがないので，CSSを打ち直す必要性がありそうです．

それが終わったらデータが綺麗に吹っ飛んだポートフォリオを再作成しつつ，やってきた事もまとめたいです( ˘ω˘ )

 

 

 

では今後ともよろしくお願いします．
