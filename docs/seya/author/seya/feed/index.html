今日、初めて白井研の卒業論文を読ましてもらいました！

読んだのは荒原一成さんの「映像多重化システムと速度変化に注目したe-sports観戦での高い一体感を生み出す演出手法の提案と開発」です。

僕はLeague of Legends をプレイしているのでe-sportsに興味があり研究テーマの参考にこちらを選ばさせていただきました。

 

内容はe-sports自体を変えていくのではなく観る側の立場になってより面白い観戦方法を作り出していくという面白い論文でした。

僕だったらこういう観戦方法がしたい、こういったことが面白そうなどとても参考になりました！

 

これで一歩研究テーマ決めに近づきましたね….!

]]> 研究室一日目！

 

白井研究室では30%ルールというものがあるらしい…

というわけで僕はどんな30%ルールになっていくか考えてみました！

 

Main Project ： 就活(30%)

Sub Project ：技術勉強(30%)

Hobby ： ゲーム(20%) + 音楽(5%) + スノーボード(5%)

other ： ブログ/遊び(10%)

 

こうなりました！

研究と就活のどちらかをメインに持ってくるか悩んだのですが、まだ研究テーマが決まっていないのでまずは研究をメインに持ってきました！！

アルバイトは今居酒屋で働いているのですが、仕事に繋げるためにプログラム系のアルバイトを経験したくてサブに入れました！

趣味はゲームでSteamを使って色々なゲームを漁っています。メインはLeague of Legends をやっていてLJLのインターンを紹介してもらえるみたいなのでとても楽しみです！

ギターは基本放置しているのですが、少しだけ触っていて友達と遊び程度に弾いています！

スノーボードを大学１年生に初めて行ってはまってしまい２年間で８回ほど滑りに行きました！運動神経はあまりよくないのですが、スノーボードはどんどん覚えてしまい、今ではジャンプなども出来るようになってしまいました！次はジャンプで回ってみたいと思います！

まぁ就活が始まるので今年は我慢しますけどね…

 

さて、これから白井研究室のメンバーとして頑張っていきますので、皆さんよろしくお願い致します！！

 

 

あ、よいお年を。
