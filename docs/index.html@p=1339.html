祝「ラブプレス＋＋」大成功＆ベータ２公開

  • 10年 ago
  • Posted in:ラブプレス, 古いお知らせ
  • 0
  • 作成者: north
  • Tags: LovePress++, Ustream, ラブプレス＋＋, 展示, 情報メディア学科, 神奈川工科大学

2010年9月17日夜に開催された東京ゲームショウ「センスオブワンダーナイト（SOWN）2010」において、白井暁彦研・服部元史研修士学生らによる新作「ラブプレス＋＋ ～俺の嫁♡にマッサージ～」(以下LovePress)が見事、大喝采のうちに発表を成功させました。発表の様子は東京ゲームショウSOWN公式Ustreamから録画をご覧いただけます。SOWNのコンセプトである「見た瞬間、聞いた瞬間、ハッとさせられる新しいゲームコンセプト」にふさわしく、また発表9作品中日本からは3作品しか存在しないという激戦の中、第1発表者として登壇した修士1年・横田真明くんとプロデューサーを務めた白井先生が会場全体を喝采と笑いの渦に引き込みました。業界各位やメディアからの反響は大きく、中でもインプレス系情報サイト「ITメディア」で取り扱われた記事は、ITメディア内でアクセスランキング1位を達成しました。並行して東京ゲームショウ2010では、TGS-SOWNパビリオン内での動画展示に加え、スクールゾーン「神奈川工科大学」ブース内にてプレイヤブル版の展示を行っております。

また、本プロジェクトで登場したキャラクターがTwitterでバーチャルに活躍しています。皆様のタイムラインに突然出没いたしますが、本人たち一生けん命なので、かわいがってあげると喜びます。
