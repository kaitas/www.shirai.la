チャンネル争いをなくしてくれる不思議な道具

みなさん、お久しぶりです。

ライターです。
ライターも、夏休みに入り、やっと落ち着きました。

実は、この夏休みの間に早くも、白井研究室の先輩方が何やら色々されているようです(￣▽￣)

この話も、また記事にしようと思います。

 

 

 

 

さて、みなさん、前回ご紹介したExPixel覚えていらっしゃいますか？
そうそう、あのフィルター越しに、テレビの画面を見ると「おやっ！？」となるあれです。

下の写真は、一見、普通の映像が流れている画面ですよね。この画面に魔法の力で、2つの映像を同時に映し・・・

 

 

 

フィルターを通して、画面を見ると・・・

 

 

 

上の写真のように、別の映像が見える、という仕組みになっています。

 

 

「でも、それって、役に立つの？」と思ったそこのあなたっ！！

えぇ、役に立ちますとも。
皆さん、こんな場面に遭遇したことありませんか？
息子「やーだぁーー！！僕、妖怪ウォッチがみーたーいー！！！」

母「いい加減にしなさい！ママだってね、福士蒼汰を拝みたいの！」

息子「やだやだやだ！」
父「ママ、大人げないよ」

母「なんですって！？あなたもこの子の味方な訳！？なによ！みんなして！大体ね、あなたが福士蒼汰みたいなイケメンだったら、全部、丸くおさまったのよ！」
父「ママだって、北川景子みたいな美人だったら、全部、丸くおさまったんだよ！」
母「はぁ！？！？一体、誰が毎日、料理して、家事して、洗濯してると思ってるのよ！」
ナレーション「ヒートアップしてゆく、夫婦喧嘩…。それを黙ってみつめるしかない息子…。どうして、こうなってしまったのか。全てのことの発端は、ただのテレビなのに…。」
母「もう離婚よ！離婚！！」
父「あぁ！お前が本気なら、こっちも本気だ！」
ナレーション「このままでいいのか…！？果たして、この後、どうなる…！」
息子「ねぇねぇ、ライターさん、このままじゃ、ママとパパが…」

ライター「大丈夫。安心して、息子ちゃん。今から、お姉さんが助けてあげるから」

息子「うん…」
ライター「えっと、こんな時は…（ガサゴソガサゴソ）
いくすぴくせるぅ〜〜（ExPixel）！

ナレーション「そう言って、ライターが取り出したのは、神奈川工科大学 白井研究室のExPixelだった」

 

 

息子「うわぁ〜！ジバニャンだ！！」

母「なによ、あなた。勝手に人の家に上がり込んで、うちのテレビのチャンネルを妖怪ウォッチにするなんて」

ライター「まぁまぁ、お母さん、このメガネをかけて、画面を見てみてください」

ナレーション「ライターから謎のメガネを受け取った、母。仕方なく、メガネをかけると、なんと・・・」

母「あらー！！福士蒼汰君じゃない！」

息子「ゲラゲラポー！　ゲーラゲラポー！」母「今日の福士蒼汰君もカッコいいわ～」

父「一体何が・・・」

ライター「実は、今までフィルターと言っていたものをレンズにした、下の写真のようなメガネを、お母さんに渡したのです」

ライター「なので、メガネをかけていない私とお父さん、それから息子ちゃんは、いくらテレビ画面をみても、妖怪ウォッチですが、メガネをかけたお母さんだけは、今ごろ、福士蒼汰君に癒されているはずです」

父「これなら・・・」

ライター「テレビで喧嘩をすることは、ないですね♪」

 

 

 

 

そう、ExPixelがあれば、家庭内のチャンネル争いをなくせるのです！

さらに、今、巷で話題を呼んでいるゲーム　Splatoon。

2人対戦の場合、通常は、一人がテレビ画面を見ながら、もう一人はゲームパッドを見ながら、プレイとなりますが、ExPixelを使えば、二人ともテレビ画面を見ながら、プレイできるのです！

 

すごいですね。

現在も、進化させるべく、日々、研究中です。

また、進化があれば、ここでも紹介したいと思います。

 

 

さて、次回は、冒頭でもお話ししたように、海外に行っていた先輩方に、インタビューしちゃいます

お楽しみに♪
