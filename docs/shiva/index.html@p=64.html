迫る魔物、闘え諸君！

最近、いいお天気が続いてますね。

では、早速、前回の答えにいってみましょう。

 

問題は、REAL SENSEを前に蛍光ペンを5本持ち、ペンを指に見立てると、はたして、REAL SENSE先輩は、どう反応するのか、でした。
実際に、蛍光ペンを持ち、REAL SENSEの前に立つと・・・？

 

反応しました！！

赤外線を出して、跳ね返ってきた赤外線で、情報を取得しているため、蛍光ペンを持っても、反応してくれる、という結果に落ち着いたのでしょう。

皆さん、いかがでしたか？REAL SENSE先輩。
こんな具合で、白井研究室がどんなことをやっているのか、をお伝えしていければ思っています。

 

 

ところで…。

みなさん、夏休みですね(^_^)

しかし、世の中には、締め切りという「魔物」がいます。大学には、期末試験という「魔物」が住んでいます。

そんな中、実は、ライター、なんと…。

先週の金曜日から土曜日にかけて「睡眠時間が…！」という経験を致しました…！！
では、何故、ライターが「睡眠時間が…っ！！」となったのか。
実は、金曜日の夜に、先輩たちとExPixelのコンテンツのデータを編集して、テストし、作成したコンテンツを実装して「よし！終わった！」と思い、時計をみたら…((((；ﾟДﾟ)))))))
そして、土曜日。

 

実は…。

土曜日 7時45分の NHK「おはよう日本」にて、白井研究室のExPixelが紹介されました。

ライター、その生放送の現場を体感すべく、朝4時起床。そして、都内へ。

今まで、朝の4時に起きることがあったでしょうか。

 

 

いや、ない！！（笑）
会場に到着すると、NHKの中継車が眩しく輝いていました。

 

 

 

そして、フリップボードに何かを書き込む白井先生。とても真剣なご様子。

 

 

 

そして、念入りに行われるリハーサル

早朝からでしたが、とても充実していて、なんだか、ライター、特に何もしてないのに、研究者になった気分でした。

 

 

さて、皆さん、こう思っていらっしゃる方、きっと多いでしょう。

「ExPixelってなにっ！？！？」と。

ExPixel、簡単に言うと、こんな感じです。

 

皆さん、よーーーくご覧ください。

画面の右上と、左下、ちょっと画面が違うように見えませんか？

右上なんだかわからないけど、左下になんか、ニュース番組のテロップみたいなのが映ってる？

これが、ExPixelです！

 

 

 

えっ？

そんな説明で分かるわけないだろ！！ですって？

 

では、来週またここでお会いしましょう。

なにが起こっているのか、というのは、次回、説明したいと思います。

今回は、とりあえず「なんか、わけわからないことが起こってる・・・！？」というのを感じていただければ大丈夫です。

 

 

ちなみに、上でご紹介したExPixel。現在、TEPIA先端技術館にて展示中です。

実際に見てみたい！という方は、ぜひ。

 

それでは、またお会いしましょう。
