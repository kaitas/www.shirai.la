Multiplex World Augmentation Display

Release Humankind from Confined Displays

Multiplex World Augmentation Display project solves the issue of sharing contents on a single large screen by augmenting three to twelve contents simultaneously. It can expand public information space and release human from confines of small displays like mobile phone by integrating Interactive media with identification technologies. Our multiplex technology started its development back in 2010 and now it's ready to test on the market. We have ScritterH for DMD projectors, ExPixel for 3DLCD, and ExField to augment 4K LCD with plastic lens. ExPixel FPGA can generate multiplex-hidden image from 2 HDMI sources like game platform. "ExPixel Generator" can easily generate multi-language presentation on PowerPoint.
