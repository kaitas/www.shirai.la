お話を頂いた時は少々不安でしたが，大変貴重な体験をさせていただけたことに感謝しています．文章を書くことや，まとめる技術，取材する能力が自分には足りていないと痛感しましたが，この経験を通して少しだけ身についたと思います．

体験や経験を文章という形でアウトプットする能力は非常に重要な能力であるので，これからも鍛えていきたいと思います．もちろんブログも！
