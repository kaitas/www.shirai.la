夏休みが始まってから2週連続でハッカソンに参加していたのでそのレポートをまとめようと思います。

 

まず8/2-3に東京工科大学にて開催された

「東北ITコンセプト　福島ゲームジャム2014」に参加してきました！！

ゲームジャムはこれで三度目、しかも最初に参加したGGJ2014と同じ会場の東京工科大学ということでめちゃめちゃ張り切って参加しました。

 

今回のテーマは「Continue」続けるという意味の英単語ですね。

私達のチームはそこから、続ける→終わらない→ゲームオーバーのないゲームというふうにつなげていきました。

最終的にずっと続くものってなんだろうとブレストをしたところ、マトリョーシカ人形に辿り着き、マトリョーシカ人形が中から出てきてどんどんどんどん世代交代(?)刷るようなゲームにしよう！という結論に。

 

そして出来たゲームがこちら。「孫量子化」です。

動画 www.youtube.com/watch?v=D8f8HnYwiT8
WebPlayer 157.7.235.138/dokuwiki/doku.php?id=team%3Ahachioji5

 

ジャンプしたり、刺にぶつかったりするたびに、なかから新しいマトリョーシカが出てきます。これを世代交代と捉え、今何代目かというスコアリングもあります。

津田はプレイヤー関係のスクリプト全般を担当しました。ここまでガッツリUnityを触ったのはこれが初めてでしたが超楽しかったです。

そして、個人的に一番びっくりだったのが、最終的に完了タスクの割合がかなり多く、全体的に完成度の高いものになったことです。

上からwill,do,done

写真で見ても、完了タスクが多い事がわかります。それだけ満足の行く出来になりました。

このゲームジャムを通してかなり成長することができたと思います。

次はOculus Game Jamかな？それまでにもっとUnity使えるようになっておこうと思いました。
