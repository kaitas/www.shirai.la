初めての研究，初めての論文執筆ということで最初は日本語の使い方を復習するところから始まりましたが，周りの方々の協力もあって無事書き終えることが出来ました．支えてくれた方々に感謝しています．

そんな私の初めての学会についてのお知らせです。
