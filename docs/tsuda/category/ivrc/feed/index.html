昨日風呂に入っている時に考えていたのだが、もっとコンテンツを深めていこうと思ってUnityでちょっとしたゲームもつくって組み合わせてみようと思った。そこで急遽スイッチを追加。そのスイッチを入れるとゲームフェイズに遷移する的なものを考え中。こういう2次元、3次元的な発想って楽しい。せっかく作ったのにそれだけにしか使えないデバイスだと意味がないからね。

 

そのあとは授業を受けて、明日のIVRCプレゼン大会に向けて研究室で名刺を作成することに。20枚刷ったが足りるだろうか…

そして5限は隔週恒例のなんちゃってTA。機材が人数分足りなかったため学生が授業を受けてる後ろで即興半田付け。うん、TAっぽい。授業も楽しく、TAとか言いながら自分がやったことないことだったのでこっそりお勉強。タメになる。

 

さて明日はIVRCプレゼン発表。残念ながらスタッフとしての参加だが、めいいっぱい楽しんでこようと思う。
