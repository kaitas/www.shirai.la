白井セミナーが終わって

全13回(初週はガイダンス、最終週はお休み)が終わって

自分の弱みが分かり、それをどう克服するかが、今後の社会に出るまで、出た後の課題となりました。

それは「モチベーション」です。

白井セミナーに配属されてから、IVRCの企画考案(大体第5回ぐらい)までは自分の意識している中のモチベーションはありました。

ですが、第6週からとなると、自習課題を忘れ、結局は他人の企画に乗るということになりました。

そこで自分の「責任感」というモノが無くなり、最後まで復調することはありませんでした。

このブログの更新もわずか2週で更新が終わり、最終週で慌てて書いています。

この自身の弱みを夏休み中に克服し、どう4年生、就活に挑むかで今後が大きく変わりそうです。

 

白井セミナー生　武田 竜平
