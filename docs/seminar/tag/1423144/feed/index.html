白井セミナーについては、自分の感じ方は苦手な部類に入ると思います。しかしこれも何かの縁なのでキッチリと苦手を克服して自分の糧にしていきたいと思います。

第１回目でも言ったように、モデリングを仕上げるのが自分の夢への一歩です。今、人形のモデリングをしています。しかしまだ人に見せれる状態ではありません。このセミナーの期間以内には仕上げたいと思っています。
