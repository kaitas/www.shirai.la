シラバス抜粋

学修目標・学修内容

コンピュータアートを含む、テクノロジーアート、メディアアートについて、国際的な歴史、作例、構造を理解し、メッセージ、表現、手法の基礎を理解し、実際の作例を鑑賞しながら、社会と制作者両側から作品性の理解を深め、方法論として獲得する。

学修成果

Webや広告、映像、エンタテイメントシステムのエンジニアとしての視点において、メディアアートを産業に活用する具体的思考習慣を身に付ける。
