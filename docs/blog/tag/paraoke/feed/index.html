神奈川工科大学情報メディア学科は、日頃の学生達の研究成果を発表する場として2011年度学生成果発表会 「Media Expo2012」を開催いたします。

開催日時：2012年3月17日（土）
開催場所：富士ソフトアキバホールとレセプションホール
