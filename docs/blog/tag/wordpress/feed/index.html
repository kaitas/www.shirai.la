本日のテーマは「大人ブログの立ち上げ方」．

WordPressをつかって，「自分が恥ずかしくないBlog」を立ち上げていきます．

 

はじめてちゃんとしたBlogを立ち上げる上での，ポイントは…

・Googleで名前を検索した時に一番上に来ることを目指すこと

会社の人事は見てますよ～，これはあなたにとっての「一次ソース」です．嘘や噂を防ぐためにもちゃんとデザインしよう．同姓同名さんと間違われていいことはないです．

・履歴書の左側をイメージ

住所を書く必要はありませんが，所属学科などは正しく書きましょう．特に高校卒業以降のアクティビティは，ぜひ．

・履歴書の右側をイメージ

趣味や頑張っていること，親しみを感じられるように．ネタに困るなら，いままで取った講義の講義名とか成績とかリストにしてみよう．何か発見があると思います．

・Twitterのアカウントなど

見られて困らないものはリンクしましょう．Pixivなどのアカウントなども「見てもらってナンボ」です．

・ネット上の自分のペルソナをデザインしよう

嘘をつく必要はありませんが，賞罰であれば「賞」は書いたほうが良いと思います．資格とか．

・感情的にならない，コントロールする．

楽しそうに書くことは大事ですが，ふざけてる，と思われると不利益です．

・大人ブログと子供ブログの違いは？

　様々あると思いますが，「隠すこと」，「ウソを付くこと」，「不明瞭・不誠実であること」，「頻繁に書かない」ことはどれも同じ事だと思います．

誠実であるためには，頻繁な更新と，深い洞察が必要です．

質の高いブログでは

「価値ある情報は，みんなに発信してこそ価値がある」

という考え方が重要です．習慣化して行きましょう．
