加藤 匠
「入社5年で体験してきたゲーム作りの現場について」

ゲーム作りの制作工程や現場の話をします。

■加藤匠 ～ 略歴／経験タイトル

同年4月 株式会社フロム・ソフトウェア 入社。『DARK SOULS Ⅱ』，『DARK SOULS Ⅱ SCHOLAR OF THE FIRST SIN』の開発に企画として参加。
