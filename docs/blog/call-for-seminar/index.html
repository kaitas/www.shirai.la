【新3年生向け】平成25年度情報メディアセミナー・白井セミナー受講者募集中

  • 7年 ago
  • Posted in:2013, 学内向け情報
  • 0
  • 作成者: shirai
  • Tags: IVRC, みんなでつくる相模原, プロジェクト, 募集, 情報メディアセミナー, 白井研究室, 相模原市, 相模原市立博物館, 知的探求散策アルバム

前期　情報メディアセミナーⅠ

エンタテイメント・バーチャルリアリティ

【身につく技術】：映像系技術，デバイス系技術，自己管理能力
人を楽しませるバーチャルリアリティシステムを企画・立案・開発し、国際学生VRコンテスト「IVRC2013」に挑戦する。IVRCは国際的なステージにおいて活躍する、数多くの研究者・クリエイターを輩出している、世界で最も歴史のある国際学生バーチャルリアリティ作品コンテストである　（https://ivrc.net/）。本セミナーでは実際にチームを編成し、企画・調査・設計・構造的作文力を習得し、プレゼンテーション審査通過、および東京大会での実機デモ発表を目座す。

 

後期　情報メディアセミナーⅡ

ソーシャルメディア・エンジニアリング

【身につく技術】：Web系技術，映像系技術，自己管理能力
テクノロジーを習得し、使いこなすことは重要であるが「その技術で社会をどう変革させ、社会に貢献するか」について考え、価値を生み出すプロセスを実践・体得することが、これからの社会をエンジニアとして確実に生き抜いていくための重要なスキルのひとつとなりえる。
本セミナーの受講者は白井研究室と相模原市立博物館とのコラボレーションプロジェクトである「みんなでつくる相模原『知的探求散策アルバム』」にサポートスタッフとして参加する。Webプログラミングやデモ開発に加え、チームで活動するため、コミュニケーション能力や、コラボレーションスキル、マネジメントスキル、加えて幅広い層のユーザにシステムを展開するスキルを得ることができる。相模原市民歓迎。
【必ずこの内容を見て応募すること】　https://blog.shirai.la/blog/2012/12/sagamihara-accepted/

【付記】

前期と後期、どちらを取ろうか迷う人は前期を受講することをおすすめします。
後期の相模原市立博物館とのコラボは前期からプロジェクトが始まっていますので、セミナーとは別に参加可能です。

その他質問があれば、Twitter(@o_ob)などでコミュニケーションをとってみてください！
