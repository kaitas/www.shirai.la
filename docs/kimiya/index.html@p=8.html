    加藤仁也、初めてのスマホとブログ

    Published by kimiya on 2016年1月18日

    こんばんは、1323026の加藤仁也です。

    初めてのブログであり、不手際もあるかもしれませんがこれからよろしくお願いいたします。

    将来の夢はゲームクリエイター。特にイラスト関係の仕事に尽きたいと思っております。

     

    今回のセミナーのため、初めてスマホを購入しました。

    購入したものは「LG　G3　Beat」、「UQmobile」というSIMカードと共に購入したものです。

    SIMとスマートフォンの別購入をやろうとして失敗してしまいまして…

    購入方法はスマートとはいえませんでした…要勉強かつ反省。

     

    慣れないスマートフォンのため、これから寝ずに勉強です。

    アプリも一から入れないといけないため、これから長くなりそう…
