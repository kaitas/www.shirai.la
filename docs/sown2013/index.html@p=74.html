SwingMasterVer1

投稿日時: 2013年7月8日 投稿者: kamiishi

アクションの一つである「うなづく」と「かしげる」の検出テストです。

[youtube]http://www.youtube.com/watch?v=fO31sHFTpo4[/youtube]

まだ開始位置の指定をしていませんが”下を向いた”という情報の取得に成功し「うなずく」動作の速度の場合にしか取得していません。
この二つを含め６個のアクションを実装予定です。

カテゴリー: 未分類   作成者: kamiishi パーマリンク
