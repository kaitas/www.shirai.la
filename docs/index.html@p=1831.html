ニコニコ超会議2にて「ニコニコメガネ」を招待展示

  • 7年 ago
  • Posted in:2013, 外部発表
  • 0
  • 作成者: shirai
  • Tags: Scritter, ニコニコメガネ, ニコニコ超会議, 展示, 情報メディア学科, 白井研究室, 神奈川工科大学

4月27,28日に開催されたニコニコ動画（株式会社ドワンゴ）が主催する、
オフラインユーザイベント「ニコニコ超会議２」（幕張メッセ・来場者10万人規模）、
ドワンゴ公式「超ニコニコ未来開発」ブースにおきまして、 白井研究室の新作「ニコニコメガネ」を招待展示しました。
https://www.chokaigi.jp/2013/booth/category/nicodevelop.html

[browser-shot url=”https://av.watch.impress.co.jp/img/avw/docs/597/889/nico14.jpg” width=”600″]

来場者数　10万3561人(去年度9万2384人)
ネット来場者数　509万4944人(去年度347万0766人)

★事後まとめと写真を追加し、事前情報を下げました。

まずはお声をかけていただいた岩城さまより。

超ニコニコ未来開発ブース担当の岩城です。以前別のイベントでScritterを拝見した際に、
「これだ！」と思いまして、今回無理をいいましてニコニコ超会議2での展示にご協力いただきました。
おかげさまでエイプリルフールにからめた事前プロモーションも含め、ニコニコのユーザのみなさんに
「未来の技術」を広く体験していただけました。ちょっと不思議な、未来の妄想のようなんだけれど、
実はもうそこにある。そんな感覚が楽しめたと思います。ありがとうございました！

株式会社ドワンゴ ニコニコ事業統括本部 プラットフォーム事業本部
岩城 進之介
